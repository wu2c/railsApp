To play, in the shell, go to 'myApp' dir, i.e. my project dir.
Hope you can run it by starting the rails server.

This application allows user sign up and log in to manage newsarticles.
All people can view newsarticle. 
But to create/edit/delete a newsarticle, user need to sign up for new user and to log in for existing users.
Only the owner of a newsarticle can make changes or delete his/her newsarticle.
Everyone can add a comment for a newsarticle.
Only the owner of a newsarticle can add/edit/delete tag(s) for such newsarticle.

This application only applys Rspec and Cucumber features for testing.
